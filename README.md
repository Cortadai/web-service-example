Este es un proyecto de demostración de **servicio web SOAP** construido con **Spring Boot 2.7.3** y Java 11, que implementa un sistema de gestión de tiendas (stores). El servicio expone un endpoint SOAP seguro (protegido con WS-Security usando UsernameToken y Timestamp) que permite consultar información de tiendas por ID, donde cada tienda contiene datos como nombre, ubicación y un gerente (manager) con email, contraseña y rol (ADMIN, USER o GUEST). El repositorio incluye datos de prueba en memoria con tres tiendas precargadas ubicadas en Dublín, Copenhague y Springfield, y utiliza un esquema XSD (stores.xsd) para definir el contrato del servicio web, generando automáticamente el WSDL correspondiente.
